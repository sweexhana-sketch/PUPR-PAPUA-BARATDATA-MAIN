LANGKAH 1: CEK NODE.JS
──────────────────────────────────────────────────────────────────
1. Tekan tombol Windows + R
2. Ketik: cmd
3. Tekan Enter
4. Di jendela hitam yang muncul, ketik: node -v
5. Tekan Enter

HASIL YANG DIHARAPKAN:
  ✓ Muncul versi seperti: v20.11.0 atau v18.x.x
  
JIKA MUNCUL ERROR:
  ✗ "node is not recognized..."
  → Node.js belum terinstall!
  → Download dari: https://nodejs.org/
  → Install, lalu RESTART KOMPUTER
  → Ulangi dari Langkah 1


LANGKAH 2: INSTALL DEPENDENCIES (HANYA SEKALI)
──────────────────────────────────────────────────────────────────
1. Buka folder ini di File Explorer:
   C:\Users\User\Downloads\PUPR-PAPUA-BARATDATA-MAIN-main

2. Di address bar File Explorer, ketik: cmd
   (Ini akan membuka CMD di folder tersebut)

3. Di jendela CMD yang muncul, ketik: npm install
4. Tekan Enter
5. TUNGGU sampai selesai (bisa 5-15 menit)
   Akan muncul banyak tulisan - ini normal!

CATATAN: Langkah ini HANYA perlu dilakukan SEKALI saja.
         Jika sudah pernah, skip ke Langkah 3.


LANGKAH 3: JALANKAN SERVER
──────────────────────────────────────────────────────────────────
PILIHAN A - PAKAI FILE .BAT (TERMUDAH):
  1. Di File Explorer, cari file: MULAI_SERVER.bat
  2. DOUBLE-CLICK file tersebut
  3. Jendela hitam akan muncul
  4. TUNGGU sampai muncul tulisan:
     "Local: http://localhost:3000/"
  5. JANGAN TUTUP jendela hitam tersebut!
  6. Buka browser (Chrome/Edge)
  7. Ketik di address bar: http://localhost:3000/webgis
  8. Tekan Enter

PILIHAN B - MANUAL (JIKA PILIHAN A GAGAL):
  1. Buka CMD di folder ini (seperti Langkah 2 poin 1-2)
  2. Ketik: npm run dev
  3. Tekan Enter
  4. Tunggu sampai muncul: "Local: http://localhost:3000/"
  5. JANGAN TUTUP jendela CMD!
  6. Buka browser, ketik: http://localhost:3000/webgis
