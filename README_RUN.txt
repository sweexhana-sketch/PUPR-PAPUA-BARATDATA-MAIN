CARA MENJALANKAN APLIKASI WEBGIS PEMPROV PAPUA BARAT DAYA
===========================================================

1. Pastikan Anda sudah menginstall Node.js.
   (Jika belum, download di https://nodejs.org/)

2. Buka Terminal / CMD di folder ini (folder PUPR-PAPUA-BARATDATA-MAIN-main).

3. Ketik perintah berikut dan tekan Enter:
   npm install

   (Tunggu proses download selesai. Ini hanya perlu dilakukan sekali di awal).

4. Ketik perintah untuk menjalankan server:
   npm run dev

5. Tunggu sampai muncul tulisan seperti:
   Local: http://localhost:3000/

6. Buka browser (Chrome/Edge) dan akses alamat:
   http://localhost:3000/webgis

CATATAN PENTING:
- Jangan tutup jendela terminal selama Anda ingin membuka web.
- Jika muncul error di browser, refresh halaman (F5).
- Jika ada masalah "Port in use", coba matikan terminal lain atau restart komputer.
